# Topdesk triage system

Create a virtual environment and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

For Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```


Setup `.env` file with

```
ANTHROPIC_API_KEY=your_anthropic_api_key
```
